# Assets Folder

Place your media files here. The following files are referenced in the default config:

## Music (assets/music/)
- `background_music.mp3` - Main background music

## Videos (assets/videos/)
- `background.mp4` - Main background video

## Images (assets/images/)
- `profile.gif` - Your profile avatar
- `owner.gif` - Owner badge
- `verified.gif` - Verified badge
- `developer.png` - Developer badge
- `partner.gif` - Partner badge

## Fonts (assets/fonts/)
- Add custom fonts here (.ttf, .woff, .woff2)

## Cursors (assets/cursors/)
- Add custom cursor files here (.png, .cur)

---

**Note:** You can use any filenames you want - just update `config.js` to match!
